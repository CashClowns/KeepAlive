Why Does Windows 11 Lock the Screen?

Windows 11 locks your screen after a period of inactivity as a security measure. The default timeout is typically set by your organization (on corporate machines) or by Windows defaults (usually 5–15 minutes on personal machines). When the screen locks, you need to re-authenticate — which is annoying if you're just reading a document, watching a training video, or stepped away for a moment.

Method 2 — Windows Power & Sleep Settings

Easy · Requires Local Admin

The standard way to change screen timeout on personal machines.

  1. Press Windows + I to open Settings
  2. Go to System → Power & Sleep
  3. Under Screen, change both "On battery power" and "When plugged in" to Never
  4. Under Sleep, set both options to Never

Note: on corporate laptops, these settings are often grayed out or overridden by Group Policy. If you can't change them, use Method 1.

Method 3 — Screen Saver Settings

Easy · Sometimes Locked by IT

Screen saver settings control the lock screen trigger separately from power settings.

  1. Press Windows + S and search for "Change screen saver"
  2. In the Screen Saver Settings window, set Screen Saver to None
  3. Uncheck "On resume, display logon screen"
  4. Click OK

Method 4 — Local Group Policy (Advanced)

Advanced · Requires Admin Rights

For Windows 11 Pro and Enterprise users with local admin access.

  1. Press Windows + R, type gpedit.msc and press Enter
  2. Navigate to Computer Configuration → Administrative Templates → Control Panel → Personalization
  3. Double-click "Screen saver timeout" and set it to Disabled
  4. Double-click "Password protect the screen saver" and set it to Disabled

This method is not available on Windows 11 Home and will be overridden by domain Group Policy on corporate machines.

Method 5 — Registry Edit (Last Resort)

Advanced · Proceed With Caution

Only use this if other methods have failed and you have admin rights.

  1. Press Windows + R, type regedit and press Enter
  2. Navigate to HKEY_CURRENT_USER\Control Panel\Desktop
  3. Find ScreenSaveActive and set the value to 0
  4. Find ScreenSaverIsSecure and set the value to 0
  5. Restart your computer
